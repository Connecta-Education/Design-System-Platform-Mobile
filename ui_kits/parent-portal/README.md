# Connecta — Parent portal UI kit

Calm guardian dashboard. Trust-first by visual design: the hero says "Nothing to flag. That's the goal." instead of a list of suspicious activity. High-signal alerts only.

## Components
- `PSidebar` — left nav, EU-residency reassurance pinned bottom.
- `PHero` — ink hero strip with this-week summary.
- `PStatTile` — bordered card with eyebrow + big number + sub.
- `PAlertCard` — nudge / pause / alert variants. Plum-on-rose for true alerts; never red.
- `PChildCard` — per-child summary row.
- `POverview` — full assembled overview screen.

## Caveats
Recreated against brand brief; no production codebase to mirror.
